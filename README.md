# banhammerv2
This is a minecraft 1.21.11 paper plugin.

This plugin allows you to grant yourself a ban hammer.
The command will be given below with instructions.

When using this plugin, Please give credits to the owner.

# Instructions

Command Usage: /banhammer 1-60 second/minute/hour/day/week/month/year.

Example: /banhammer 1 hour.
The example will result in giving you a banhammer that bans a player for 1 hour when hit.

The player must be in survival mode.
